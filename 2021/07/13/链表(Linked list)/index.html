链表(Linked list)

链表
数据结构
发布日期:   2021-07-13
更新日期:   2021-07-13
文章字数:   647

简介(Introduction)

链表是一种物理存储单元上非连续、非顺序的存储结构，数据元素的逻辑顺序是通过链表中的指针链接次序实现的。链表由一系列结点（链表中每一个元素称为结点）组成，结点可以在运行时动态生成。每个结点包括两个部分：一个是存储数据元素的数据域，另一个是存储下一个结点地址的指针域。

描述(Description)

由于结构体加指针类型的链表在开辟内存空间时运行速度很慢，这里只介绍利用数组模拟的链表(静态链表)。

示例(Example)

单链表(Singly Linked List)

单链表常见的操作：

  1. 初始化

    代码实现(Code)

    int head, e[N],ne[N], idx;
//初始化链表：
void Init()
{
 head = -1;
 idx = 0;
}
  2. 向表头插入一个数：

示例(Example)

代码实现(Code)

void add_to_head(int x)
{
    e[idx] = x;
    ne[idx] = head;
    head = idx++;
}
  1. 在第k个数后面插入一个数

代码实现(Code)

void add(int k, int x)
{
    e[idx] = x;
    ne[idx] = ne[k];
    ne[k] = idx++;
}
  1. 删除第k个插入的数的后面一个数：

示例(Example)

代码实现(Code)

void del(int k)//注意传入时要传k - 1;
{
    ne[k] = ne[ne[k]];
}
  1. 删除头结点

代码实现(Code)

head = ne[head];

双链表(Dual Linked List)

双链表没有单链表常用，因为各种操作稍微复杂。
直接给出代码和一点解释：

代码实现(Code)

  1. 初始化
  int l[N],r[N],e[N];//左右指针和值
int cnt=0;
void Init()
{
    //0表示左端点，1表示右端点
    r[0]=1,l[1]=0;//一开始左右互指
    cnt=1; //cnt从1开始
}

  1. 在第k个插入的点右边插入值为x的点(右插)(传入k + 1)

    void add_r(int k,int x)
{
 e[++cnt]=x;//计数器加一并且存值
 r[cnt]=r[k];
 l[cnt]=k；
 l[r[k]]=cnt;
 r[k]=cnt; //注意一定要先改变r[k]的左指针，再改变r[k]
 //一旦先改变r[k]，那么r[k]的左指针就失效了。

  2. 在第k个插入的点左边插入值为x的点(左插)(传入k + 1)
    直接调用

    add_r(l[k],x);
//第k个点左边插入等价于在第k个点的左边一个点的右边插入。
//当然，模仿又插再写一个也行。

  3. 删除第k个插入的点(传入 k + 1)

    void del(int k)
{
 r[l[k]]=r[k];
 l[r[k]]=l[k];
}

这些都是链表的基本操作，尤其是单链表要熟练掌握。
